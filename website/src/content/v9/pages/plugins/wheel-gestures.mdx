# Wheel Gestures

<Admonition type="danger">
  This plugin does not currently support v9. For updates, please refer to the

<RepositoryLink href="https://github.com/xiel/embla-carousel-wheel-gestures" />

This plugin adds **mouse and trackpad wheel support** to Embla Carousel, allowing users to navigate slides using scroll gestures.

---

## Options

Below follows an exhaustive **list of all** `Wheel Gestures` **options** and their default values.

---

### wheelDraggingClass

<ApiMetaData
  firstRow={['Type', 'string']}
  secondRow={['Default', 'is-wheel-dragging']}
/>
<PrismHighlight
  language="ts"
  hideLabel
  code={`{ wheelDraggingClass: 'is-wheel-dragging' }`}
/>

Choose a classname that will be applied to the container during a wheel gesture. Pass an empty string to opt-out.

---

### forceWheelAxis

<ApiMetaData
  firstRow={['Type', 'string']}
  secondRow={['Default', 'undefined']}
/>
<PrismHighlight
  language="ts"
  hideLabel
  code={`{ forceWheelAxis: 'x' | 'y' }`}
/>

Force an axis on which to listen for wheel events. Choose scroll axis between `x` and `y`. Useful if you want to slide horizontally when scrolling vertically or vice versa.

---

### target

<ApiMetaData
  firstRow={['Type', 'Element']}
  secondRow={['Default', 'undefined']}
/>
<PrismHighlight
  language="ts"
  hideLabel
  code={`{ target: document.querySelector('.embla__viewport') }`}
/>

Specify the element that should be observed for wheel events.
